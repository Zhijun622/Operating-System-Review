## 三、综合应用题（本大题4小题，每小题10分，共40分）

### 1题：进程同步问题（10分）

某水果店有一个能放1个水果的果盘，爸爸专门往盘子里放苹果，妈妈专门往盘子里放橘子，儿子专门等着吃盘子中的苹果，女儿专门等着吃盘子中的橘子。只有盘子空时，爸爸或妈妈才能放一个水果；只有盘子中有自己需要的水果时，儿子或女儿才能从中取出。

**要求：**
1. 用信号量和PV操作描述上述四个并发进程的同步关系。（6分）
2. 写出所需信号量的定义及初值。（4分）

### 2题：银行家算法（10分）

某系统中有5个进程P0、P1、P2、P3、P4和3类资源A、B、C，各类资源的数量分别为10、5、7。在T0时刻的资源分配情况如下表所示：

| 进程 | Max(A,B,C) | Allocation(A,B,C) |
|------|------------|-------------------|
| P0   | 7, 5, 3    | 0, 1, 0           |
| P1   | 3, 2, 2    | 2, 0, 0           |
| P2   | 9, 0, 2    | 3, 0, 2           |
| P3   | 2, 2, 2    | 2, 1, 1           |
| P4   | 4, 3, 3    | 0, 0, 2           |

**要求：**
1. 计算Need矩阵和Available向量。（3分）
2. 判断T0时刻系统是否处于安全状态？如果安全，请给出一个安全序列。（4分）
3. 在T0时刻，若进程P1发出资源请求Request1 = (1, 0, 2)，系统能否立即分配？请说明理由。（3分）

### 3题：处理机调度（10分）

有四个作业A、B、C、D，它们的到达时间和运行时间如下表所示：

| 作业 | 到达时间 | 运行时间 |
|------|----------|----------|
| A    | 8:00     | 2.0小时  |
| B    | 8:30     | 1.0小时  |
| C    | 9:00     | 0.5小时  |
| D    | 9:30     | 0.3小时  |

**要求：**
1. 分别采用先来先服务（FCFS）算法和短作业优先（SJF）算法进行调度，写出调度顺序。（2分）
2. 对于FCFS算法，计算每个作业的周转时间和带权周转时间，并计算平均周转时间和平均带权周转时间。（4分）
3. 对于SJF算法，计算平均周转时间和平均带权周转时间。（3分）
4. 比较两种算法的性能。（1分）

### 4题：存储管理（10分）

某请求分页系统，页面大小为1KB，为某进程分配了3个物理块，其页表如下：

| 页号 | 块号 | 状态位 | 访问位 | 修改位 |
|------|------|--------|--------|--------|
| 0    | 5    | 1      | 1      | 0      |
| 1    | 8    | 1      | 0      | 1      |
| 2    | -    | 0      | 0      | 0      |
| 3    | 2    | 1      | 1      | 1      |
| 4    | -    | 0      | 0      | 0      |

**第一部分：地址映射（4分）**

该进程依次访问以下逻辑地址（十进制）：2100, 1024

要求：
1. 计算逻辑地址2100对应的页号和页内偏移。（2分）
2. 计算逻辑地址2100对应的物理地址。（2分）

**第二部分：页面置换（6分）**

假设该进程后续依次访问页面序列：1, 2, 3, 4, 1, 2, 5, 1, 2（从左到右）

当前内存中的页面为：0, 1, 3

要求：
1. 分别采用FIFO算法和LRU算法进行页面置换，画出置换过程。（4分）
2. 计算两种算法的缺页次数和缺页率。（2分）
