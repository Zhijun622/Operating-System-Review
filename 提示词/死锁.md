# 模块：死锁 (Deadlock)

# 目标
这是必考大题之一。请结合上传的【死锁.md】习题、PPT 和教材，生成一份以**死锁避免（银行家算法）**为核心的复习笔记，并针对习题分析常考知识点。

# 任务要求
1. **原理与条件**：详细解释死锁的**四个必要条件**，并结合【习题 10】强调**静态分配**不属于必要条件。
2. **预防与避免对比**：
    * **死锁预防**：对比破坏四个必要条件的四种策略（结合【习题 2, 4, 7】分析），并讨论它们的资源代价（R利用率）。
    * **死锁避免 (重点)**：详细解析安全状态的概念，以及银行家算法作为避免策略的原理（结合【习题 3, 4】）。
3. **银行家算法实战**：使用清晰的**序号列表**描述银行家算法的判断流程和安全性算法的执行步骤（必考大题）。

# 习题生成要求
1. **生成相似选择题**：根据【死锁.md】中的**习题 8**（最少资源数问题），生成 **2 道** 类似的选择题，测试学生对“不发生死锁所需资源数”的计算和理解。
2. **生成大题**：生成 **1 道** 银行家算法的应用题，要求给出资源总量和进程需求，要求判断当前状态是否安全。
