4. # 模块：存储管理 (Storage Management)

   # 目标
   这是必考大题之一。请根据【复习大纲】中“四、存储管理”的要求，结合上传的 PPT 和教材，生成一份以**离散存储与置换算法**为核心的复习笔记。

   # 任务要求
   1. **实存管理**：对比连续分配（固定/可变分区）和离散分配（分页、分段、段页式）各自的特点、优缺点和碎片类型（内碎片/外碎片）。
   2. **离散地址映射**：以**请求分页存储管理**为核心，详细解释**逻辑地址**如何通过**页表**映射到**物理地址**的机制，并说明 TLB/快表的作用。
   3. **置换算法详解 (必考大题)**：
       * 详细对比 **OPT、FIFO、LRU** 这三种置换算法的原理和性能。
       * **计算模型**：整理缺页率、有效访问时间 (EAT) 的计算公式。

   # 习题生成要求
   请基于这份笔记内容，生成 **2 道** 综合计算题（如：给定一个页面访问串和内存块数，要求分别计算使用 FIFO 和 LRU 算法时的缺页次数和缺页率）。
