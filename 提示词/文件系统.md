# 模块：文件系统 (File System)

# 目标
请根据【复习大纲】中“五、文件系统”的要求，结合上传的 PPT 和教材，生成一份以**结构、辅存管理和目录管理**为核心的复习笔记。

# 任务要求
1. **文件结构对比**：重点解析并对比三种物理结构（连续文件、链接文件、索引文件）的优缺点，特别是**索引文件**（单级、多级、混合索引）的实现逻辑。
2. **辅存管理**：详细解释**三种**空闲空间管理方法（空闲表、空闲链表、位示图法）的原理，并给出其适用场景。
3. **目录管理**：解释“目录项 (FCB)”和 “i-node” 的概念及其在文件共享中的作用（如符号链接）。

# 习题生成要求
请基于这份笔记内容，生成 **3 道** 选择题，题目应测试对三种文件物理结构（链接/索引/连续）的查找效率和存储效率的对比。
